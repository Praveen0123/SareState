Screen Layout

Realtor can setup and manage the brokers/agents from this screen. This screen consists of Add broker/agent & filter buttons and Grid View which displays the complete data of broker/agents with ability to edit and delete.

Add Broker/Agent - Modal Popup

Name:* Text Box
Role:* Drop down
Builder:* multi select drop down
Project:* drop down
Mobile Number:* Text box
Email:* Text box
Address:* Text box
Status:* drop down
Save and Cancel button

Filter Fields
Name: Keyword Search
Role: drop down
Builder: drop down
Project: drop down
Status: drop down
Search and Clear buttons

Grid View - Column Fields

Name, Role (either broker/agent), Builder Assigned, Project Assigned, Contact Number, Email, Address, Status (either Active or Inactive), Actions (Edit & Delete icons)
