Screen Layout

Admin can setup and manage the builders from this screen. This screen consists of Add builder & filter buttons and Grid View which displays the complete data of builders with ability to edit and delete.

Add Builder - Modal Popup

Builder Name:* Text Box
Project:* drop down
Mobile Number:* Text box
Email:* Text box
Address:* Text box
Status:* drop down
Save and Cancel button

Filter Fields
Builder Name: Keyword Search
Project: drop down
Status: drop down
Search and Clear buttons

Grid View - Column Fields

Builder Name, Project Assigned, Builder Contact Number, Builder Email, Address, Status (either Active or Inactive), Actions (Edit & Delete icons)
